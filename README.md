# Habit Insight Client

Habit Insight is a React TypeScript frontend for a full-stack habit tracking and analytics application.

The app allows users to create habits, complete them with mood notes, track progress history, and analyze long-term consistency through dashboard statistics and charts.

This frontend is connected to a Node.js / Express / MongoDB backend API.
